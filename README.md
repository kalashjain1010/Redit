


## Technology used

**Client:** React,Next.js,Chakra-UI Redux, Styled-Components

**Server:** Firebase


## Run Locally

Clone the project

```bash
  git clone https://github.com/kalashjain1010/Redit.git
```

Go to the project directory

```bash
  cd Redit
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
